## Streamlit101-workshop
- Interactive Streamlit app: [app.py](app.py)
- Companion notebook: [Streamlit_workshop.ipynb](Streamlit_workshop.ipynb)
- Sample data: [dataset.csv](dataset.csv)

### How to run the Streamlit app locally
- Install dependencies: `pip install -r requirements.txt`
- Launch: `streamlit run app.py`
- The app opens in your browser at http://localhost:8501

### How to use the notebook
- Open [Streamlit_workshop.ipynb](Streamlit_workshop.ipynb) in VS Code or Jupyter
- Run cells top-to-bottom to explore data preparation and modeling steps
- Export any insights or models into the Streamlit app for interactive sharing

### Deploy to Streamlit Community Cloud
- Push this repo to GitHub
- On streamlit.io → Deploy app → point to `app.py` on the `main` branch
- Set secrets (if any) in the Streamlit Cloud dashboard
- App will install from [requirements.txt](requirements.txt) automatically
